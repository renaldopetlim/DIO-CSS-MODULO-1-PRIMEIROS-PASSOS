Seletores CSS

Bem-Vindo!

No CSS nós possuímos os:

  • Seletores por Tipo: Busca elementos por uma tag HTML específica.
  • Seletores por ID (#): Busca elemento através do atributo "id".
  • Seletores por Classe (.): Busca elementos através do atributo "class".
  • Seletor Universal (*): Seleciona todos os elementos HTML.
  • Seletores por Atributo ([atrib]): Seleciona elementos que possuem um atributo específico em nosso documento HTML e conseguimos também buscar atributos com um valor específico.

Agrupamento de Seletores

Podemos aplicar as mesmas regras CSS para vários seletores diferentes. Para isso, podemos separar os seletores utilizando uma vírgula (,).

Assim, não precisamos ficar copiando e colando os mesmos códigos para todos, nós podemos fazer um só bloco de CSS e reutilizar as mesmas regras!
